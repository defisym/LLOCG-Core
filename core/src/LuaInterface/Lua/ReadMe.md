# Lua

Download pre-compiled lua binaries from [here](https://sourceforge.net/projects/luabinaries/files/5.4.2/).

If the link failed, find a new link from [here](https://luabinaries.sourceforge.net/).

The file structure should be like this:

```plaintext
./
├── include/
│   ├── lua.h
│   ├── lua.hpp
│   └── ...
├── x64/
│   ├── lua54.dll
│   └── lua54.lib
└── x86/
    ├── lua54.dll
    └── lua54.lib
```
